# MojiLog (모지로그)

> 제작자의 반려동물 '모지'의 일상을 공유하고 팬들과 소통할 수 있는 게시판형 블로그
<img src="https://user-images.githubusercontent.com/108676517/206964157-cc86293d-568d-465c-9e11-6d29319b637d.png"  width="700" height="500">



## 프로젝트 개요
- **개발 기간** : 2022.08.16 ~ 2022.09.07 (약 4주)
- **개발 인원** : 1명 (개인 프로젝트)
- **개발 환경**
  - 운영체제  : Window10
  - 개발 방식 : Model 1
  - 개발 툴   : Apache Tomcat v9.0, Eclipse, MySQL
  - 개발 언어 : Java, JavaScript, HTML5, CSS3, SQL, JSP
  - Open API : Kakao 주소 API
- **주요 기능** : Model 1 방식을 사용하여 회원가입, 로그인, 게시판CRUD 기능 구현, 파일 업로드 및 다운로드

## 느낀점
> 개발을 진행하면서 DB 테이블 컬럼 수정이 잦았고, 이에 따라 DTO, DAO 파일도 함께 수정하면서 시간이 많이 걸렸다. 추후 프로젝트 진행 시 초기 DB 설계를 꼼꼼히 잘 해야겠다는 것을 느꼈다.
